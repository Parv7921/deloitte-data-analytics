# Deloitte Data Analytics Virtual Experience

## Project Overview

This repository contains my completed work for the Deloitte Data Analytics Virtual Experience Program on Forage. The project focuses on transforming raw business data into meaningful insights through data visualization and equality analysis using Tableau and Microsoft Excel.

The project demonstrates practical data analytics skills by building an interactive dashboard, analyzing operational data, and evaluating workplace equality metrics to support data-driven decision-making.

---

## Business Problem

Organizations rely on data analytics to monitor operational performance and identify business improvement opportunities. In this virtual experience, the objective was to analyze operational telemetry data, build an interactive Tableau dashboard, and examine workforce equality data to generate meaningful business insights.

---

## Project Objectives

- Analyze operational telemetry data.
- Build an interactive Tableau dashboard.
- Perform equality analysis using Microsoft Excel.
- Present insights that support business decision-making.
- Demonstrate practical data analytics and visualization skills.
